пятница · еженедельно

костёр

круг, шеринг без непрошеных советов. всё что родилось — в общее пространство. social glue, не контент

1–2 в неделю

протагонист

unblocking

час на один твой затык. приходишь со своим зависанием — уходишь с движением. фасилитатор ведёт, ты работаешь

alignment

кто я сейчас, куда двигаюсь, что приоритет, что отвлечение. артефакт-контракт + AI коуч. якорный формат, к которому возвращаешься

спринты

что делают внутри

живые кейсы участников. без имён — приватный чат остаётся приватным. то, что там родилось, участники сами выносят наружу когда готовы
здоровье
медицинская ОС для себя и мамы
ситуация: анамнез + генетика + лабы + заключения — везде по отдельности, собрать невозможно
AI: Claude + GitHub, консилиум врача и нутрициолога, все данные в одной среде
результат: дашборд с рисками, упакован в open-source — кто хочет, раскатывает у себя
смысл: можно не ждать системы здравоохранения — собрать свою
здоровье
интеграция сенсоров и генетики
ситуация: Oura + CGM + 23andMe лежат по отдельности, никто не видит картину целиком
AI: Claude + Codex, morning summary в telegram-бот, данные по еде и активности
результат: нашёлся lactose intolerance со ссылками на статьи — GPT до этого говорил обратное
смысл: AI не просто читает данные — ищет противоречия и сверяется сам
саморазвитие
коуч-промпт для внутренней работы
ситуация: сложный момент 2-летней давности, до сих пор триггерит, тело держит
AI: коуч-промпт в Claude — выгрузил ситуацию, мысли, чувства
результат: «в моменте стало легко» — появилось взрослое место, не вина. пошла настоящая работа
смысл: AI как отражение, которое подсвечивает то что сам не видишь
саморазвитие
личный коуч на ценностях и паттернах
ситуация: 20+ недель записей, видны паттерны саботажа, нужна система сверки с собой
AI: кастомный коуч-промпт — встроены ценности, IFS, схема-терапия
результат: Claude задаёт уточняющие вопросы, берёт в джорналинг, выдаёт гипотезы на неделю
смысл: «очень кайфую, какой приятный и любопытный может быть процесс»
работа
unified os для проектов
ситуация: операционка комьюнити + 5 личных проектов = хаос окон и вкладок
AI: Claude OS + VSCodium — KB, задачи, терминал в одном экране. Codex берёт задачи сам
результат: 11-часовая сессия co-thinking с моделью, оркестрация без ручного контроля
смысл: AI — не «помощник», а соавтор, с которым вместе думаешь и строишь
работа
open-source voice kit на Rust
ситуация: нужна быстрая транскрипция локально, на маке, без облака и токенов
AI: переписал inference с Node на Rust, интеграция через Claude Code
результат: 18× быстрее Whisper на Apple Silicon, один бинарник 20 MB, open-source для всех
смысл: инструмент, сделанный для себя, работает и для других — это и есть комьюнити
