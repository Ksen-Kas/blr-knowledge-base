кейсы участников

здоровье
Маша Куликовская · врач
ситуация: анализы разбросаны по клиникам разных стран, врачи не видят полную картину
AI: команда агентов-врачей с полярными позициями — консервативный vs прогрессивный
результат: полная медицинская история в одном месте. маме нашла альтернативу статинам
смысл: LLM — отражение твоего мышления. важно добавлять полярные позиции намеренно
здоровье
Тоня Жукова · предприниматель
ситуация: 7 лет повышенного холестерина, ни один врач не спросил про Lp(a)
AI: Automoy — 18 агентов (скауты, критики, статистики, фактчекеры)
результат: нашёлся Lp(a) как причина. Portfolio diet → анализы улучшились
смысл: система ищет не ответы на вопрос — она ищет interconnections с учётом контекста
здоровье
Влад Куклев
ситуация: 8 лет попыток выстроить систему здоровья, программы рассыпались при первом пропуске
AI: AI-тренер в Telegram — Gemini Studio + Claude Code + Whoop API
результат: 5 недель подряд 5–6 тренировок без пропусков и волевых усилий
смысл: агент знает твою историю, пишет сам когда нужно. это не чат — это кто-то кто помнит всё
здоровье
Катя Кирилова · CEO
ситуация: здоровье как проект — нужен не один эксперт, а команда
AI: 8 агентов-коучей: longevity, спорт, красота, питание + Garmin API
результат: персональная команда экспертов 24/7, автоматизированные тренировки
смысл: AI позволяет собрать команду под себя — без найма и ожидания
здоровье
Лёша Евдокимов
ситуация: хотел понять свой метаболизм — не абстрактно, а на реальных данных
AI: CGM 16 дней + Claude Code, данные каждые 5 минут, жена — контрольная группа
результат: визуализация метаболизма которую не даёт ни одна клиника
смысл: AI снижает барьер между «хочу понять» и реальным инструментом исследования
саморазвитие
Макс Янукович · философ техники
ситуация: вопрос не «как использовать AI», а «как оставаться собой рядом с AI»
AI: Claude как собеседник, не инструмент. диалог как метод исследования
результат: язык для разговора о субъектности и AI — без тревоги и хайпа
смысл: AI как зеркало для мышления, а не инструмент автоматизации
